## Hands-On
- Extracted raw CSV data into a distributed Spark environment.
- Cleaned and Cast column data types for analysis.
- Grouped feature columns into a single vector.
- Trained multiple regression algorithms (Linear Regression, Decision Tree, Random Forest).
- Logged all model runs into the MLflow tracking UI.
- Compared performance to identify the Random Forest as the best model.
- Built a spark ML Pipeline to automate the workflow.
- Predicted purchases on test data.
- Validated results by displaying actual vs. predicted values.
